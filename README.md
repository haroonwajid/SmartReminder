## Getting Started

1. **Clone the repo:**
   ```sh
   git clone https://github.com/haroonwajid/SmartReminder.git
   ```
2. **Open in Xcode:**
   Open `Reminder-app.xcodeproj` in Xcode (iOS 16+ recommended).
3. **Build & Run:**
   Select an iOS Simulator or your device and hit Run.
